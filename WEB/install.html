How to Install Logicytics

To install and set up Logicytics, follow these steps:

  • Install Python: Download from the official website. Select "Add Python to PATH" during installation.
  • Install Dependencies: Run pip install -r requirements.txt in your terminal.
  • Run Logicytics: Run python Logicytics.py -h in your terminal to open the help menu.

Prerequisites

  • Python: Requires Python 3.8 or higher. Download from the official website.
  • Dependencies: Install with pip install -r requirements.txt.
  • Administrative Privileges: Run with admin privileges for certain features.
  • System Requirements: Tested on Windows 10 and 11. Not compatible with other OS.
  • Command Line Knowledge: Basic understanding recommended.
